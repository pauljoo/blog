# 脚本和语法

## 第一行

`#!`告诉系统同一行上紧跟在它后面的那个参数是用来执行本文件的程序。

```shell
#!/bin.sh
```

## 语法

### 变量

使用变量之前通常并不需要事先为它们做出声明。
变量赋值时，等号两边不能有空格。
通过在变量名前加一个$符号来访问它的内容。

```shell
a=123
echo $a
```

#### 环境变量

当一个shell脚本程序开始执行时，一些变量会根据环境设置中的值进行初始化。这些变量通常用大写字母做名字，以便它们和用户在脚本程序里定义的变量区分开来。

|环境变量|说明|
|-|-|
|$HOME|当前用户目录|
|$PATH|搜索地址|
|$PS1|命令提示符|
|$PS2|二级提示符|
|$IFS|输入域分隔符|
|$0|shell脚本的名字|
|$#|传递给脚本的参数个数|
|$$|shell脚本的进程号|

#### 参数变量

如果脚本程序在调用时带有参数，一些额外的变量就会被创建。

|环境变量|说明|
|-|-|
|$1,$2,...|脚本程序的参数|
|$*|列出所有|
